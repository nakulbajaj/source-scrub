Source Scrubber

This is a tool to remove personal information from source code or other text.

Input the text from which you would like personal information removed into the Source Code box below, and then any personal information you would like to have removed. You can read more about how the tool works (and why you can trust it) at the bottom of the page.

Other personal information to search for and redact from the source code.

We are not storing anything you enter into this tool, including your personal information. In fact, the tool runs entirely client side, in your web browser, meaning none of the information is being sent to a server or leaving your browser in any way. This tool is hosted with GitHub Pages, from the open source repository found here: https://github.com/changkm/source-scrub/.
